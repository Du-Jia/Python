# Python中函数的使用
* *Python中函数的定义时函数名作为一个变量，这个变量存储的是整个函数对象*
* *Python中函数参数不用声明类型，同时参数可以接受任意类型数据*
### 定义函数：
```

  #function body
  return name list bing divided with ','
```

* 注意：Python中return语句可以返回多个值，当返回多个值时，把变量用逗号`,`隔开，这些变量将会被打包成一个元组返回

